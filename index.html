Metalithic

Design and front-end development

Metalithic is an art studio from Perth, Western Australia. They desired a way to show off their handcrafted mixed-media pieces on the web. The website allows customers to view individual pieces through the gallery, learn more about the fascinating process behind the pieces, and enquire about purchasing.

The Metalithic site is built using a variety of modern web technologies. Stacey, a PHP-based flat-file CMS, runs on the back end. I use Sass as the preprocessor, which is compiled and minified by popular task runner Grunt.

Craftoid

Design, front- and back-end development

Craftoid is a web application for the popular indie game Robocraft. It allows users to share their awesome creations made with the in-game editor. Some of the main features include:

  • Sign in and Sign up
  • Competitions with voting on submissions
  • Ability to 'favourite' other users' creations
  • Browse creations by time, popularity, Tier, or category
  • Submit comments

The application is built on Ruby on Rails, with a PostgreSQL database. I used heroku for deployment, and the entire application and related assets are managed with Git version control.
